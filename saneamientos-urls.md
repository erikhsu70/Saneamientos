# URLs Saneamientos — enlaces internos

Añade aquí las URLs canónicas del sitio (servicios, categorías de blog, landings) para usarlas en la fase de linking antes de publicar.

## Formato sugerido

```markdown
## Servicios / páginas clave
- https://www.ejemplo.com/servicio-1 — descripción breve
- https://www.ejemplo.com/servicio-2 — descripción breve

## Blog (posts recurrentes)
- https://www.ejemplo.com/blog/slug-post — tema
```

*(Rellenar cuando tengas dominio y estructura definitiva.)*
