# Saneamientos (WordPress)

Publicación de artículos en Markdown → WordPress (mismo flujo que `legalpin-wp`).

- **Guía técnica:** [README-wordpress.md](README-wordpress.md)
- **Credenciales:** copia `.env.example` a `.env` y rellena `WP_URL`, `WP_USER`, `WP_LOGIN_PASSWORD`.
- **Plantilla de post:** [article-template.md](article-template.md)
- **URLs para enlaces internos:** [saneamientos-urls.md](saneamientos-urls.md) (rellenar con el mapa del sitio)
